# 🎮 Tic-Tac-Toe  

## 📝 About  
This is a **simple 2-player Tic-Tac-Toe game** built with **React**, using **vanilla CSS** for styling. The game offers a fun and interactive experience for two players to enjoy!

---

## ✨ Key Features  
- 🧑‍🤝‍🧑 **Player Customization:** Players can **choose and edit their names**.  
- 🎯 **Turn Tracking:** A **pulsating border** highlights the active player's name to keep track of turns.  
- 📝 **Move Logging:** Every move is logged, displaying the **player’s name** and the **field coordinates** chosen.  
- 🏆 **Smart Game Logic:** Game logic compares the **current state** with predefined **winning combinations** stored in a file.  
- 🎉 **Game Over Pop-Up:** A **special GameOver component** announces the winner or displays an **"It's a draw!"** message when appropriate.

---

## ⚛️ React Concepts Used  
- 🔄 **State Management:**  
  - Separate states for **players' data** and **turn tracking**, ensuring a dynamic log display.  
- 📊 **State Updates:**  
  - Ensures updates are based on the **previous state**, accommodating React’s asynchronous state updates.  
- ✍️ **User Input & Two-Way Binding:**  
  - Utilizes React’s **onChange event** for seamless input handling.  
- ✏️ **Immutable State Updates:**  
  - Handles updates to objects and arrays immutably, ensuring stability.  
- 🛠️ **State Lifting:**  
  - State is lifted to the **closest common ancestor** component to enable sharing between relevant components.
